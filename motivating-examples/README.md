# Motivating examples
This is the sandbox for thinking through how the language should act.

## syntax

The details of the syntax have not been worked out, the goal here is to
brainstorm about the capabilities of the language.
The convention adopted here:
    
  * python-like. indentation, not braces. However tabs prohibited as indentation.
  * python keywords
  * builtins and other reserverd words start with `.'
