# dayhoff
This is a hobby project; it may never be functional.

What I want: a high level language for describing algorithms which:
  1. can be automatically converted to C (or some other fast language),
  2. separates algorithmic logic from the data model

See the [motivating-examples](./motivating-examples/README.md) subdirectory

# Dedication
[Margaret Dayhoff](https://en.wikipedia.org/wiki/Margaret_Oakley_Dayhoff)
was an incredibly brilliant scientist. Using her training in chemistry, mathematics, and biology, she founded the field of bioinformatics..
